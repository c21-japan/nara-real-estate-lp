よくあるご質問

お客様からよくいただく質問にお答えします

売却の相談から契約までどのくらいの期間がかかりますか？ +
通常は1〜3ヶ月程度です。広告開始から2週間〜1ヶ月以内に購入希望者が現れるケースが多いです。
売却の際にはどのような書類が必要ですか？ +
特に何もご用意は必要ございません。必要な書類は当社でご案内・サポートいたします。
売却活動はどのように行われますか？ +
ポータルサイト掲載、業者間ネットワーク（レインズ）活用、チラシ配布、現地案内などを併用します。センチュリー21の全国ネットワークも活用し、幅広く購入希望者を探します。
売却後の引き渡しまでの流れを教えてください +
契約→ローン承認→決済日調整→残金受領と同時に鍵引渡しという流れになります。当社が最後までサポートいたします。
売却時にかかる仲介手数料はいくらですか？ +
売買価格×3％＋6万円（＋消費税）です。成功報酬のため、売却が成立しなければ費用はかかりません。
売却益にかかる税金はどのくらいですか？ +
所有期間5年以内は短期譲渡（税率約39％）、5年超は長期譲渡（税率約20％）となります。詳しくは税理士をご紹介することも可能です。
売却時にかかるその他の費用はありますか？ +
印紙税、抵当権抹消費用、登記費用、引っ越し費用などがかかります。詳細は無料査定時にご説明いたします。
住宅ローンが残っている場合でも売却できますか？ +
可能です。売却代金でローン残債を一括返済し、抵当権を抹消します。残債が売却価格を上回る場合もご相談ください。
売却活動中に住みながらでも大丈夫ですか？ +
可能です。内覧の際は事前に日程調整を行い、生活感を抑えるためのアドバイスも実施いたします。
売却価格はどのように決まりますか？ +
一般的に近隣の成約事例、市場動向、物件状態を総合的に判断して査定しますが、当店は不動産売却に特化した店舗なので、高額売却のご提案ができます。
売却が決まらなかった場合、広告費は請求されますか？ +
原則として成約に至らなければ仲介手数料・広告費は不要です。安心してお任せください。
急いで売却したい場合の方法はありますか？ +
センチュリー21グループによる直接買取や投資家への即時紹介、各企業様への紹介が可能です。最短3日での現金化も可能です。
